# Command Center - Business Operations Hub

## Overview
This is the master navigation portal for all operational dashboards of Nicolas Gonzalez's business operations. The hub serves as a centralized command center with a dark luxury theme.

## Features
- **Dark Luxury Theme**: Background #0f1117, cards #1a1d28, gold accent #b8956a
- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Interactive Cards**: Hover effects, smooth animations, and click feedback
- **Status Indicators**: LIVE (gold), COMING SOON (blue), PLANNED (gray)
- **Business Portfolio**: Grid of all 7 businesses with type indicators

## Technical Details
- **Fonts**: Inter (body) + Playfair Display (headings) from Google Fonts
- **Icons**: Font Awesome 6.4.0
- **Animations**: CSS transitions and keyframe animations
- **JavaScript**: Minimal interactive effects (ripple, parallax)
- **CNAME**: hub.jackiespanties.com

## Deployment
The site is ready to be deployed to any static hosting service (Netlify, Vercel, GitHub Pages, etc.). The CNAME file is included for custom domain setup.
